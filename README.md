# Satora Website

> Marketing and documentation website for Satora

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Cloudflare Workers (via OpenNext)

## Getting Started

```bash
yarn install
yarn dev
```

The site runs at [http://localhost:3000](http://localhost:3000)

## Deployment

```bash
yarn deploy
```

Requires Cloudflare account access. Wrangler config is in `wrangler.jsonc`.
