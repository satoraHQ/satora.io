## TypeScript SDK (Recommended)

Our SDK is pure TypeScript - no WASM or native dependencies. It works in browsers, React Native, and Node.js out of the box.

## Iframe

Embed Satora Swaps directly into your application - no SDK integration required. Already used by [Arkade Wallet](https://arkade.money), [Alby Hub](https://albyhub.com), and more.

```html
<!-- Add your reflink from the Developer Portal to track swaps -->
<iframe
  src="https://lendaswap.com?ref=YOUR_REFLINK"
  width="100%"
  height="700"
  frameborder="0"
  allow="clipboard-write"
/>
```
