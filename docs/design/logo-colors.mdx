## Motto

> **Make Bitcoin Move.**

This is the Satora tagline. Use it in page titles, social media, and marketing materials.

---

## Logo

### Wordmark

The Satora logo is a pure text wordmark: **satora** followed by a small lime-green dot. No icon is used. The dot scales proportionally with the font size (0.3em).

### Logo Variants

| Variant               | Background    | Usage                  |
| --------------------- | ------------- | ---------------------- |
| Black text + lime dot | White / light | Navbar, headers, press |
| White text + lime dot | Black / dark  | Navbar, headers, press |

### The Green Dot

The green dot is the core element of the Satora brand identity. Used on its own, it serves as the **profile picture**, **favicon**, and **app icon** across all platforms. Whenever a square or circular brand mark is needed, use the dot alone.

    <span style={{ fontSize: "13px", color: "#737373" }}>Light mode dot</span>

  <div style={{ padding: "1.5rem 2rem", background: "#0a0a0a", borderRadius: "16px", border: "1px solid #222", display: "flex", alignItems: "center", gap: "1rem" }}>
    <img src="/assets/logo/satora_dot_darkmode.svg" alt="Satora dot (dark mode)" width="48" height="48" />
    <span style={{ fontSize: "13px", color: "#999" }}>Dark mode dot</span>
  </div>
</div>

| Asset           | Usage                                                       |
| --------------- | ----------------------------------------------------------- |
| Green dot alone | Profile pictures, favicons, app icons, social media avatars |
| Full wordmark   | Website headers, documentation, press materials             |

<Warning>
  **Never use the lime-green wordmark on a white or light background.** The lime-green variant (`#c2e821`) is only permitted on black or dark backgrounds. On white or light backgrounds, always use the pure black version.
</Warning>

### Logo Usage in Code

```tsx
/* The SatoraLogo component - use this everywhere */
<span className="font-bold text-black dark:text-white select-none">
  satora
  <span
    className="inline-block rounded-full bg-lime-400 align-baseline"
    style={{ width: "0.3em", height: "0.3em", marginLeft: "0.05em" }}
  />
</span>;
```

The dot is sized at **0.3em** relative to the font, so it scales automatically at any size. It sits on the **baseline** of the text.
