## Usage Guidelines

### Do

- Use the text wordmark "satora." with the lime dot on a clean, uncluttered background
- Use the green dot alone for profile pictures, favicons, and app icons
- Maintain adequate spacing around the wordmark
- Use black text on light backgrounds and white text on dark backgrounds
- Let the dot scale with the font size (0.3em)

### Don't

- Use a separate icon/image next to the wordmark - the logo is text-only
- Alter the dot color or proportions
- Place the wordmark on busy or low-contrast backgrounds
- Add effects like drop shadows, gradients, or outlines
- Use the old "Lendasat" branding - the company name is **Satora**
