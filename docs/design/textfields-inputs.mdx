## Input States

All inputs follow the same interaction pattern:

| State        | Effect                                | Timing       |
| ------------ | ------------------------------------- | ------------ |
| **Default**  | Subtle border, neutral background     | -            |
| **Hover**    | Border lightens slightly              | `150ms` ease |
| **Focus**    | Lime border + ring glow               | `150ms` ease |
| **Invalid**  | Red border + ring                     | `150ms` ease |
| **Disabled** | `opacity(0.5)`, `cursor: not-allowed` | -            |

```
/* Tailwind classes for all inputs */
border border-gray-200 dark:border-white/10
bg-white dark:bg-white/5
rounded-xl outline-none transition-colors duration-150
focus:border-lime dark:focus:border-lime-light focus:ring-2 focus:ring-lime/20
```

Three sizes available:

| ------ | ------------- | ---------------- |
| **sm** | `h-9` (36px)  | `text-[13px]`    |
| **md** | `h-11` (44px) | `text-sm` (14px) |
| **lg** | `h-12` (48px) | `text-[15px]`    |

```
/* Standard input */
h-11 px-4 rounded-xl border border-gray-200 dark:border-white/10
bg-white dark:bg-white/5

focus:border-lime dark:focus:border-lime-light focus:ring-2 focus:ring-lime/20
outline-none transition-colors duration-150
disabled:cursor-not-allowed disabled:opacity-50
```
