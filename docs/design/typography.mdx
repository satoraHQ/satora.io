## Spacing

| Token               | Value                           |
| ------------------- | ------------------------------- |
| Container max-width | `72rem` / `max-w-6xl`           |
| Page padding        | `px-6` / `sm:px-8` / `lg:px-12` |
| Section vertical    | `py-32` (128px)                 |
| Card padding        | `p-7` (28px)                    |

### Border Radius

| Element            | Radius               |
| ------------------ | -------------------- |
| Buttons, Navbar    | `rounded-full`       |
| Cards, Code window | `rounded-2xl` (16px) |
| Inputs, Icon boxes | `rounded-xl` (12px)  |

## Icons

Heroicons v2 Outline (`react-icons/hi2`). Always outline, never filled.

| Context       | Size    |
| ------------- | ------- |
| Feature cards | 28px    |
| Buttons / nav | 14-16px |

Feature card icons are always lime. Nav/UI icons are gray. Chain icons use official colored SVGs from `/assets/chains/`.

## Code Syntax

Two-color theme: **lime** (structure) + **Bitcoin orange** (values).

| Token             | Light     | Dark      |
| ----------------- | --------- | --------- |
| Keywords          | `#7a8a0e` | `#c2e821` |
| Functions         | `#7a8a0e` | `#a3c410` |
| Strings           | `#e88332` | `#f7931a` |
| Types / Variables | `#c07415` | `#f0a94e` |
| Comments          | `#9ca3af` | `#6b7280` |
